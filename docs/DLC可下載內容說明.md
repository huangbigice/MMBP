# DLC（Downloadable Content）可下載內容說明

**文件編號**：DLC-001  
**版本**：1.0  
**生效日期**：________  
**適用產品**：智慧股票投資決策支援系統（MMBP + FMFA）

---

## 一、文件目的

本文件說明本系統提供之可下載內容（DLC）種類、格式、取得方式與使用條款，供客戶於採購、導入或授權使用時參考。

---

## 二、DLC 分類總覽

| 類別 | 說明 | 對象 |
|------|------|------|
| **技術文件** | API 規格、架構說明、部署指南 | 技術人員、整合開發者 |
| **報表與匯出** | 回測報告、分析結果匯出 | 投資人、研究人員 |
| **資料與模型** | 歷史資料匯出、模型版本資訊（非原始模型檔） | 依授權條款 |
| **前端資源** | 靜態建置檔、資產包 | 部署人員 |

---

## 三、可下載內容清單

### 3.1 技術文件類

| 項目 | 格式 | 內容摘要 | 取得方式 |
|------|------|----------|----------|
| **API 規格書** | PDF / OpenAPI JSON | 端點、參數、回應結構、錯誤碼 | 官網文件區／客戶入口 |
| **部署與安裝指南** | PDF / Markdown | 環境需求、安裝步驟、設定說明 | 客戶入口／技術支援 |
| **模型版本說明** | Markdown | `model_version`、`strategy_version`、訓練區間、變更日誌 | 隨系統或 `GET /api/v1/model-info` 對應文件 |
| **SLA 與可用性說明** | PDF | 服務等級、可用性目標、責任上限 | 服務條款頁／合約附約 |

### 3.2 報表與分析匯出類

| 項目 | 格式 | 內容摘要 | 取得方式 |
|------|------|----------|----------|
| **回測報告** | JSON / CSV | 權益曲線、年化報酬、波動率、最大回撤、夏普比率等 | API 回傳或前端匯出 |
| **技術指標資料** | CSV / JSON | 日期、OHLC、成交量、MA、RSI、EMA、KD、MACD 等 | `GET /api/v1/stock/{symbol}/indicators` 回傳 |
| **預測結果摘要** | JSON | 類別機率、系統／技術／基本面評分、建議文字 | `POST /api/v1/predict` 回傳 |
| **審計日誌** | JSONL | 依合約提供之審計紀錄（含日期、參數摘要、結果摘要） | 依授權由營運方匯出提供 |

### 3.3 資料與模型相關

| 項目 | 格式 | 內容摘要 | 取得方式 |
|------|------|----------|----------|
| **模型版本資訊** | JSON | `model_version`、`strategy_version`、`model_effective_date` | `GET /api/v1/model-info` |
| **歷史股價與指標** | CSV / JSON | 依 API 參數取得之區間資料 | 透過 API 或批次匯出（依授權） |
| **訓練與回測用腳本** | Python / 文件 | 訓練流程、指標計算邏輯說明 | 依商業授權或客製合約 |

> **注意**：預訓練模型檔（`.pkl` 等）及原始訓練程式碼之提供，需另訂授權條款，一般不列入標準 DLC。

### 3.4 前端資源類

| 項目 | 格式 | 內容摘要 | 取得方式 |
|------|------|----------|----------|
| **FMFA 前端建置檔** | 靜態檔案（HTML/JS/CSS） | `npm run build` 產生之 `dist/` 內容 | 部署包下載／CDN |
| **前端原始碼** | 依授權 | React 元件、樣式、API 呼叫層 | 依授權條款 |

## 四、取得與交付方式

### 4.1 線上取得

- **官網文件區**：技術文件、API 規格、FAQ
- **客戶入口**：登入後可下載授權範圍內之 DLC
- **API 直接回傳**：資料類 DLC 可經 API 即時取得（JSON/CSV）

### 4.2 離線交付

- **郵寄／檔案傳輸**：依合約提供之報告、審計日誌、部署包等
- **光碟／隨身碟**：特殊需求或資安要求時，可另議離線交付方式

### 4.3 更新頻率

| DLC 類型 | 更新頻率 | 說明 |
|----------|----------|------|
| API 規格 | 隨版本發布 | 重大 API 變更時更新 |
| 模型版本說明 | 隨模型／策略升級 | 見 `MODEL_VERSION.md` |
| 報表與資料 | 即時／依請求 | API 回傳為即時計算結果 |
| 靜態文件 | 不定期 | 依內容變更需要 |

## 五、使用條款與限制

### 5.1 一般原則

- 所有 DLC 之使用須遵守**服務條款**及**授權合約**。
- 未經書面同意，不得將 DLC 轉授權、再散布或用於本合約範圍外之用途。
- 涉及個人或機敏資料之 DLC，須符合個資法及雙方約定之資安規範。

### 5.2 資料類 DLC

- 金融資料可能源自第三方，客戶使用時須自行注意相關授權與合規。
- 歷史資料與預測結果**僅供研究與決策輔助**，不構成投資建議；投資風險由使用者自負。

### 5.3 技術文件與程式碼

- 技術文件供內部整合與維運使用。
- 原始碼之提供、修改權與再授權範圍，以個別授權條款為準。
